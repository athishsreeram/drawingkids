
# 🎨 DrawingKids

A fun, colorful, and easy-to-use drawing web app designed for kids to unleash their creativity.  
No sign‑up, no ads – just a clean canvas and simple tools.

👉 **Live demo:** [athishsreeram.github.io/drawingkids](https://athishsreeram.github.io/drawingkids/index.html)

## ✨ Features

- **Free drawing** with mouse or touch (finger/stylus)
- **Multiple colors** – pick from a kid‑friendly palette
- **Adjustable brush size** (small, medium, large)
- **Clear canvas** button to start over
- **Save drawing** as a PNG image (download)
- **Undo / Redo** – easy mistake fixing
- **Responsive layout** – works on tablets, laptops, and desktops
- **No external dependencies** – pure HTML/CSS/JS

---

## 🖥️ How to Use

1. Open the [live demo](https://athishsreeram.github.io/drawingkids/index.html) or run locally (see below).
2. Choose a color from the palette.
3. Click/tap and drag on the canvas to draw.
4. Adjust brush size using the slider or buttons.
5. Click **Clear** to erase everything.
6. Click **Save** to download your artwork as an image.
7. Use **Undo** / **Redo** to go back/forward in your drawing steps.

## 🧪 Local Development

If you want to run or modify the project on your own computer:

```bash
git clone https://github.com/athishsreeram/drawingkids.git
cd drawingkids
```

Then open `index.html` in your browser – no build step required.

### File structure (typical)
```
drawingkids/
├── index.html      # main page structure
├── style.css       # styling and responsive design
├── script.js       # drawing logic, event handlers, undo/redo
└── README.md       # this file
```
